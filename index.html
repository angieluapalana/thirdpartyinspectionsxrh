1Purpose

This SOP covers how we prepare City of Atlanta third-party inspection reports after Khaled completes a site visit.

Right now, our job stops at preparing the report. We fill it out, save it in the correct project folder, and send it to Khaled — Khaled handles the actual city submission unless he tells us otherwise.

Side-by-side so it's easy to see exactly what shifts once Khaled activates direct city submission. Steps 1–3 stay the same either way — the difference starts at step 4.

3Information Needed From Khaled

Before starting a report, confirm all of the following are available. If anything is missing or unclear, check with Khaled before finalizing.

  • Project / job address
  • Contractor or client company name
  • General contractor name
  • Building permit number
  • Trade permit number
  • Inspection type
  • Inspection code
  • Inspection result
  • Any notes or special instructions

4Correct Form to Use

Use the official colored City of Atlanta Third-Party Inspection Report form — the PDF version, whenever possible.

⬇ Download Official Form (PDF) — atlantaga.gov

Sourced from the City of Atlanta's Third-Party Inspections page. If the city updates the form, re-confirm this link still points to the current version.

AVOID Don't use the Google Docs version if its formatting looks different or distorted from the official form.
CONFIRM If the project folder has multiple versions, use whichever one Khaled has confirmed as the correct city form.

5Main Form Rules

Building permit

The building permit number always goes at the top of the report — even for electrical, mechanical, or plumbing inspections.

Trade permit

The trade permit must match the actual inspection type:

Inspection TypePermit to Use
ElectricalElectrical permit
MechanicalMechanical permit
PlumbingPlumbing permit
Building / FramingBuilding permit

6Inspection Date

City of Atlanta reports need to go out within 2 days of the inspection.

Use the inspection or report date Khaled provides.

STOP Never guess the date — not even if the site visit clearly happened earlier or the date seems unclear. Confirm with Khaled instead.

9Combining Inspections

Some inspections can go in one report — but only if Khaled confirms they were completed together on the same visit.

Examples:

  • Electrical ditch + electrical slab
  • Mechanical final + electrical final + plumbing final
STOP Never combine inspections on your own judgment — Khaled's confirmation is required every time.

10Building Final Rule

RULE Building final must always be its own, separate report. Never combine it with mechanical final, electrical final, plumbing final, or any other inspection type.

For building final, list all related permits in the report, including:

  • Building permit
  • Electrical permit
  • Mechanical permit
  • Plumbing permit
  • Sprinkler / fire permit, if applicable

11Fire / Sprinkler Notes

We typically don't handle fire inspections — those go through the city or fire marshal directly.

That said, sprinkler or fire permit details may still need to appear on building final reports.

CONFIRM Confirm with Khaled before adding any fire or sprinkler permit info, unless he's already given clear instructions for that report.

12Signature & Supporting Pages

Usually only the first page of the report needs updating per inspection.

The remaining pages hold company details, insurance, certifications, and authorization documents. Khaled provides these or lets us know when they need updating — and they may need to be refreshed yearly.

13Before Sending the Report

Run through this before sending anything to Khaled. Click an item to check it off (for your own tracking on this page — not saved).

  • Correct city form used
  • Formatting is clean and not distorted
  • Client / contractor name entered
  • Job address entered
  • General contractor entered
  • Building permit listed at the top
  • Correct trade permit entered
  • Correct inspection code marked
  • Correct result selected
  • Building final separated, if applicable
  • Signature and date included
  • File saved in the correct project folder

14File Naming Format

Use this format for every saved report:

YYYY-MM-DD–Job Address–Inspection Type–Permit Number

Example:

2026-06-18 – 340 East Wesley Ferry – Electrical Ditch and Slab – BE123456
